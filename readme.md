PRS Scheduling App (Internal)

This repository contains the source code for the Philly Rock & Soul scheduling and operations application maintained by Lutine Management Associates.

This project is intended for internal use by authorized users only.

Using the App

If you are a band member or staff user:

Access the app using the official link provided by the administrator

Create an account / log in using the email address you use to communicate with the band

If you cannot access your schedule after logging in, contact the administrator

Support

For assistance with access, account setup, or scheduling questions, contact the administrator.
